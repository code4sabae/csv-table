# csv-table
csv-table / tsv-table HTMLElement ES module  

# How to use
```
<script type="module" src="https://code4sabae.github.io/csv-table/csv-table.mjs"></script>
<csv-table>
id,name
1,なまえ
2,名前
3,"ナ
マエ"
</csv-table>
```
